# 티스토리 바이럴 깔때기 구조 설계

목적: 모두의 클로드 강의 결제 전환
전략: 찐탱 정보글로 신뢰 쌓기 → 백링크로 연결 → 최종 결제 링크

---

## 구조도

```
[1단계] 유입 글 — 키워드 잡기 / 광고색 제로
  └─ Claude Skills 실전 활용법 ✅ 완료
  └─ Claude Routines 사용법
  └─ Claude Projects 세팅법
         ↓ 백링크
[2단계] 신뢰 글 — 실사용 증명
  └─ "나는 이렇게 쓴다" 비개발자 실제 사례
  └─ Claude로 반복 업무 없앤 후기
         ↓ 백링크
[3단계] 전환 글 — 학습 필요성 환기
  └─ Claude 제대로 쓰려면 뭘 배워야 하나
  └─ 혼자 배울 때 vs 커리큘럼으로 배울 때
         ↓ 백링크
[최종] 결제 링크
  └─ 모두의 클로드 강의 랜딩페이지
```

---

## 원칙

1. 유입 글은 100% 정보성 — 광고색 제로, CTA 없음
2. 백링크는 자연스럽게 ("이 주제는 별도 글에서 다뤘습니다" 식)
3. 강의 언급은 전환 글에서만 — 그 전까지 절대 노출 안 함

## 백링크 연결 계획

- Skills 글 → Routines 글, Projects 글 연결 예정
- Routines 글 → Skills 글 역링크
- 실사용 후기 글 → 유입 글 전체 연결
- 전환 글 → 신뢰 글 + 강의 랜딩페이지 연결
